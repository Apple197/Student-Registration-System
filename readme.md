# Student Registration System



The Student Registration System is a web-based application designed to manage student records in a simple and organized way.
This system allows users to add, update, and delete student information while ensuring that the data remains saved even after refreshing the page.
The project focuses on creating a clean and user-friendly interface for handling student data efficiently.
---



## How It Works

1. The user fills out the student form
2. After submission

   * Data is checked for valid input
   * A new record is created
3. The record is displayed instantly on the page
4. Users can

   * Edit the record
   * Delete the record
5. All records are saved in the browser, so they are not lost after refresh



## How to Run the Project

1. Clone the repository
2. Open the folder
3. Open **index.html** in your browser



### Technologies Used

* HTML – structure of the page
* CSS – design and layout
* JavaScript – functionality and logic
